25-second demo: a numerical question auto-routed to the right unit, solved with the unit's own formula, arithmetic re-verified with sympy, sources shown verbatim, per-unit formula sheet.

🧭
Auto-routing

Subject, topic and query type detected from the question alone. Students pick only college + year, once.

📖
Verbatim sources

Answers come from isolated per-unit vector stores; matched sections are re-read from the authoritative Markdown, equations intact.

✓
Verified arithmetic

Numerical answers are re-computed with sympy and chip-marked as verified — or flagged when they don't match.

📐
Formula sheets & quizzes

Every display equation in a unit on one sheet; five-question practice quizzes generated from the unit's own content.

⚡
Semantic cache

Repeat questions (cosine ≥ 0.95 within a unit) answer instantly from cache.

🛡️
Hallucination control

The LLM is restricted to classification, bounded generation and verification — never open-ended recall.

Why is this a demo and not the live app? GitHub Pages hosts static files only. Wildflow needs a Python backend, ~2.5 GB of embedding models (e5-large + CLIP), per-unit Qdrant vector stores and a server-side Gemini key — it runs on any machine with Python and ~6 GB free RAM. Setup instructions are in the repository README.
